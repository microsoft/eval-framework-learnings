# evalproj

Minimal Python project scaffold created by GitHub Copilot.

Requirements
- Python 3.8+

Quickstart

1) Create a virtual environment (PowerShell):

.\.venv\Scripts\Activate.ps1

2) Install dependencies:

pip install -r requirements.txt

3) Run the package from the installed entry point (after editable install) or module:

python -m evalproj

Or using the script defined in pyproject (after install):

evalproj --name Alice

Running tests

pip install -r requirements.txt
pytest

Development

- Source code is under `src/evalproj`
- Tests are under `tests`
